# 快速开始

欢迎来到 ClaudeCode 源码分析文档。

## 关于本项目

本项目基于 Claude Code 的还原源码（reconstructed from source maps），深入分析其架构设计、模块划分和核心实现。

## 环境要求

- **Bun** >= 1.3.5
- **Node.js** >= 24.0.0

## 本地运行

```bash
# 进入文档目录
cd mydocs

# 安装依赖
bun install

# 启动开发服务器
bun run dev

# 构建
bun run build

# 预览构建结果
bun run preview
```
